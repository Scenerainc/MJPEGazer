## Getting started:

> Optional
>
> ```sh
> python3 -m venv .venv
> ```

```sh
python3 -m pip install -r requirements.txt
```

```sh
python3 main.py
```

> alternatively
>
> ```sh
> export FLASK_APP=main.py
>
> python3 -m flask run
> ```
